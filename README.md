# 🎓 Smart Flashcards Pro

Современное геймифицированное веб-приложение (SPA) для эффективного изучения иностранных слов и расширения словарного запаса. В основу проекта заложены алгоритмы адаптивного интервального повторения и умного тестирования знаний.

## ✨ Ключевые особенности

* **🧠 Алгоритм «Умных уроков» (Smart Session):** Система автоматически формирует персональный набор карточек для каждой тренировки. В приоритет выводятся слова, у которых подошёл срок интервального повторения, и проблемные слова с низким коэффициентом успешности.
* **⚡ Экзамен на вылет (Test Out):** Если вы уже хорошо знаете слово, вам не придётся учить его с нуля. Безошибочное прохождение экспресс-спринта из 4 хардкорных упражнений подряд (включая ввод руками) мгновенно отправляет слово в вечный архив 6-го уровня.
* **🔍 Интеграция с MyMemory API:** Автоматический перевод английских слов на русский язык в один клик. Система не только находит перевод, но и подгружает живой контекстный пример употребления предложения вместе с его переводом.
* **🎯 Дофаминовый трекер прогресса:** Минималистичная панель активности визуализирует ваши дневные рекорды с помощью интерактивных кубиков активности (`🟩`). При перевыполнении нормы цепочка кубиков аккуратно трансформируется в числовой счётчик уроков.
* **💾 Безопасное резервное копирование:** Встроенные функции экспорта и импорта базы данных в формате JSON через Blob-объекты. Данные бесшовно дедуплицируются при слиянии со старыми бэкапами.
* **🎨 Премиальный дизайн:** Чистый интерфейс с пастельно-неоновыми градиентами уровней, кастомными комбобоксами и плавными CSS 3D-переворотами карточек.

## 📁 Структура проекта

Проект спроектирован по модульному принципу (Separation of Concerns), что обеспечивает лёгкое масштабирование:

```text
├── index.html          # Разметка интерфейса и модальных окон
├── theme.css           # Глобальные переменные и базовые шрифты
├── layout.css          # Каркас приложения, фильтры и панели управления
├── components.css      # Стилизация 3D-карточек, бейджей и тегов категорий
├── modals-lesson.css   # Стили всплывающих окон и тренировочного бокса
├── storage.js          # Логика LocalStorage, бэкапов и истории активности
├── api.js              # Модуль интеграции с внешним API перевода и Web Speech API (озвучка)
├── lesson-builder.js   # Движок интеллектуального распределения карточек по урокам
├── lesson-validator.js # Валидация ответов, расчёт интервалов и зацикливание ошибок
├── lesson-engine.js    # Управление игровым циклом и шагами прогресса внутри урока
├── ui-manager.js       # Рендеринг динамических компонентов и сетки на главном экране
└── app.js              # Главная точка входа и связующие прослойки (Application Router)
```

## 🚀 Быстрый старт

Проект написан на чистом JavaScript (Vanilla JS) и не требует сборщиков или установки тяжёлых зависимостей.

1. Склонируйте репозиторий на локальный компьютер.
2. Откройте папку проекта в **VS Code**.
3. Установите расширение **Live Server**.
4. Нажмите кнопку **Go Live** в правом нижнем углу редактора.
5. Приложение автоматически откроется в браузере по безопасному адресу `http://127.0.0.1:5500`.

## 🛠️ Технологический стек

* **Frontend:** HTML5, CSS3 (Flexbox, Grid, 3D Transforms, CSS Variables)
* **Логика:** JavaScript ES6+ (Async/Await, Web Components Logic, Set, Splice)
* **Хранилище:** Web Storage API (LocalStorage)
